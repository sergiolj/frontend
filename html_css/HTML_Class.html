HTML Class Atributo

É usado para especificar uma classe para um elemento HTML, sendo que multiplos elementos podem compartilhar a mesma classe. O atributo classe é comummente usado para apontar para um nome de classe em uma stylesheet e pode ser utilizado também pelo JavaScript para manipular elementos específicos.

Sintaxe de uso

Para criar uma classe use qualquer nome usando o atributo CLASS=" " no elemento, tomando cuidado que classes são CASE SENSITIVE. Dentro do CSS a classe deve ser indicada com a inserção de um ponto (.) antes de seu nome seguido por aspas {} contendo as atribuições.

London

London is the capital of England.

Paris

Paris is the capital of France.

Tokyo

Tokyo is the capital of Japan.
