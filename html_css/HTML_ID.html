Atributo ID em HTML

O atributo Id é usado para especificar um identificador único para um elemento de HTML. NÃO PODE EXISTIR MAIS DE UM ELEMENTO COM O MESMO ID EM UM DOCUMENTO HTML. O id é usado para apontar para um estilo específico em uma stylesheet e também pode ser utilizado para manipular elementos específicos com JavaScript. A sintaxe para a definição de ids em CSS é # seguido do nome do id e aspas {}.

Diferença CLASSE X ID

Uma classe pode ser utilizada com multiplos elementos HTML enquanto um id só pode ser utilizado por um elemento. Os IDs também são utilizados para criação de BOOKMARKS no corpo do HTML no caso de páginas muito extensas como capítulos de um artigo ou texto longo.
