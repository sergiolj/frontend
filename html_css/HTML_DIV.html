Elemento SPAM

O SPAM é um elemento de HTML usado para destacar parte de um texto ou uma parte do documento, ele não requer qualqer atributo, mas é comum o uso de atributos de identificação como CLASS E ID para posterior uso com CSS ou atributos de STYLE. O SPAM é um atributo de nível INLINE, ou seja, só utiliza o espaço necessário para sua inserção. CASO nao sejam usados atributos visualmente o spam não sofre alteração em relação ao texto comum.

Usando Grid

O Grid é um módulo de layout do CSS que oferece um modelo de organização baseada em grid com linhas e colunas dispensando o uso de Float e de posicionamento. Similar ao Flexbox com a vantagem de poder organizar individualmente cada uma das linhas e colunas. Similar ao Flexbox e ao Float o Grid usa um container DIV externo e outros vinculados internamente. Difere apenas do uso do inline-block que não apresenta o container externo.
