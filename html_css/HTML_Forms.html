HTML Forms

O elemento HTML Form é um container para diferentes elementos HTML usado para coletar informações do usuário. Essas entradas geralmente são ennviadas para um servidor para serem processadas. São elementos comuns no container form: text fields, checkboxes, radio buttons, submit buttons, etc.






Elementos mais usados

O elemento INPUT é o mais usado no container FORM e pode ser usado para coletar informações de forma varidada dependendo do tipo de atributo que será utilizado. São atributos comuns ao INPUT: TEXT, RADIO, CKECKBOX, SUBMITR, BUTTON

Atributos do Elemento FORM

action

O primeiro dos atributos é o ACTION que define qual ação será executada quando o form foi enviado através do botão SUBMIT. Ele pode enviar para um arquivo PHP do tipo server-script side.
"form action="/action.php".
Se a informação action for omitida a ação é enviada para a página corrente.


target

Especifica onde será exibida a resposta do Submit.
"form action="/action.php" target="_blank"
As opções podem ser _blank (nova página), _self (mesma página), _parent (no frame onde se encontra o form), _top (corpo inteiro da janela) ou framename (em um iframe específico)
