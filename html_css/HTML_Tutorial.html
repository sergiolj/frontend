Título pirncipal - O que é HTML?

HTML significa Linguagem de Marcação de Hypertexto e é a linguagem padrão para criação
de páginas web, descreve a estrutura da página e consiste em uma série de elementos
que informam ao navegador como exibir o conteúdo.

Os títulos dentro de uma página web são classificados por impontância dos números 1-6

Seu uso com essa estrutura é importante porque os mecanismos de busca usam os headings
para indexar a estrutura das páginas web. Assim eles devem refletir a estrutura da página. Os headings possuem tamanho padrão mas podem ser modificados por atributos de style.
