Conceitos de tipos de Exibiçao de Elementos em HTML - Block e Inline

Todos os elementos em HTML possuem um valor padrão de exibição. Os dois valores mais comuns são Block e inline.

Elementos de nível Block

Os elementos desse tipo começam em nova linha e o browser adiciona automaticamente um espaço de margem antes e depois do elemento. Um elemento tipo Block usa toda a largura disponível se esticando da esquerda para a direita. Os mais comuns são o h1,h2.. P e o DIV.

Esse é um exemplo de div usado para um parágrafo. Lembrando que DIV significa divisão em HTML e serve para separar uma parte do texto. A quantidade de elementos do tipo Block nível é grande no HTML. Forms, OL UL Listas, HR, DIVisões, Tabelas elementos Semânticos como header, main, footer, entre outros
